## bounding-box-project

Учебный мини‑проект: IoU + простой гибридный NMS + визуализация боксов на изображении.

### Запуск демо

Через скрипт (работает без установки пакета):

```bash
uv sync
uv run python scripts/demo_nms.py --show
```

Или как модуль (как в “реальном” проекте, после установки проекта в окружение):

```bash
uv sync
uv run python -m bounding_box_project --show
```

Выходные картинки сохраняются в `outputs/`.

Если хочешь сохранить “витринные” картинки для документации, укажи:
`--out-dir docs/images`
